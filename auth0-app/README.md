# auth0-app
- auth0 golang tutorial
    - https://auth0.com/docs/quickstart/webapp/golang
- use mux as a router
- use negroni as a middleware

## Getting Started
- create auth0 apps
- set auth0 id to .env
- run application

```$xslt
$ go run server.go main.go
```

- access localhost
    - http://localhost:3000
